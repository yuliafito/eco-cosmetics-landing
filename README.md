# Eco Cosmetics Landing Page
This project showcases a clean and responsive landing page for Witch of Mountain Winds, a brand focused on eco-friendly cosmetics and handcrafted candles. The website introduces the brand’s mission and values, presents product categories, shares the creators’ story, and helps users explore and connect with the brand.

## Live Demo
Experience the live website: [Eco cosmetics](https://yuliafito.github.io/eco-cosmetics-landing/)

## Design Reference
Figma Design: [Eco cosmetics](https://www.figma.com/design/Fz588JKGuPS2Bk21De4KE5/Brand-of-eco-cosmetics-_FE-students?node-id=21779-2&p=f&t=9X4UM1Sun37hVfvl-0)

## Technologies Used
- **HTML5** – Semantic page structure
- **SCSS** – Modular styling
- **BEM methodology** – Structured and maintainable CSS
- **Responsive Design** – Support for mobile, tablet, and desktop devices

## Getting Started
Clone the repository:
```bash
git clone https://github.com/yuliafito/eco-cosmetics-landing.git
cd eco-cosmetics-landing
```
Install dependencies:
```bash
npm install
```
Run the project locally:
```bash
npm start
```

## Features
- **Responsive Design:** Optimized for screens at 320px, 640px, 1280px, and 1440px
- **Semantic HTML:** Accessible and SEO-friendly page structure
- **Smooth Hover Effects:** Animated transitions on buttons, links, and interactive elements
- **Slide-out Navigation Menu:** Burger menu with animated slide-out effect
- **Clear Page Layout:** Intuitive and well-structured content organization
